Design Authenticity Auditor

Identify lazy AI-generated layouts and get concrete fixes to make any site feel human-engineered.

How to use

  1. 1
    Enter the URL of the site you want to audit in the field below.
  2. 2
    Open the site in a separate tab so you can inspect it while you work through the checklist.
  3. 3
    Rate each criterion — toggle Human if the site does it well, Slop if it fails.
  4. 4
    Read the fix suggestions that appear below the checklist for every item you mark as Slop.

The score updates live. Hit Reset Form to start a fresh audit on a different site.

1. Typography & Content Layout

Typography & Content Layout questions

Visual Scale & Sizing Hierarchy

Human: Proportional font scale with clear H1/H2/H3 relationships. Slop: Messy sizes, no discernible typographic hierarchy.

Text Line Length

Human: Sub-800px text columns — comfortable reading width. Slop: Paragraphs stretch across the full screen width.

Padding & Whitespace Uniformity

Human: Intentional, consistent margins and padding. Slop: Crammed elements or awkward accidental empty spaces.

2. Interactions & Behaviours

Interactions & Behaviours questions

Interactive Feedback States

Human: Clear hover, active, and focus styles on every control. Slop: Static links with no visual feedback on interaction.

Responsive Window Adaptation

Human: Reflows cleanly on mobile viewports. Slop: Broken grids, text clipping, or overlapping fields on resize.

Logical Component Completeness

Human: Fully built functional flows. Slop: Empty dropdowns, unclickable tabs, or dead search inputs.

3. Visual Design & Asset Integrity

Visual Design & Asset Integrity questions

Icon Style Consistency

Human: Unified stroke weights and styles throughout. Slop: Mixed outlined, filled, and generic shapes together.

Graphics & Asset Intent

Human: Contextual, branded images that serve the content. Slop: Ultra-glossy, irrelevant AI-generated stock illustrations.

Alignment Precision

Human: Pixel-grid snapping throughout. Slop: Visually off-centre text inside boxes or misaligned buttons.

4. Content & Copy Quality

Content & Copy Quality questions

Writing Voice & Specificity

Human: Distinct brand voice, concrete facts, named examples. Slop: Vague superlatives — "revolutionary", "cutting-edge", "seamless".

Placeholder & Filler Content

Human: All sections have real, purposeful content. Slop: Lorem ipsum, placeholder images, or "Coming soon" blocks.

Social Proof Authenticity

Human: Real testimonials with full name, role, and photo. Slop: Generic avatars + AI-written praise with no attribution.

5. Performance & Technical Craft

Performance & Technical Craft questions

Page Load & Asset Optimisation

Human: Fast LCP, lazy-loaded images, minified assets. Slop: Huge unoptimised images, render-blocking scripts, slow TTFB.

Accessibility Basics

Human: Alt text, ARIA labels, full keyboard navigability. Slop: Missing alt attributes, low contrast, no visible focus ring.

Favicon & Meta Completeness

Human: Custom favicon, og:image, accurate meta description. Slop: Default browser icon, missing social previews, generic title tags.

Recommended Tools

The stack that separates considered design from AI-generated defaults.

Design & Prototyping

Figma Industry-standard UI/UX design and prototyping. Component libraries, auto-layout, and dev handoff built in. Framer Design-to-production site builder with real code output. No-compromise animations and interactions. Webflow Visual CSS editor that generates clean semantic markup. CMS and e-commerce built in. Penpot Open-source Figma alternative. Self-hostable, SVG-native, works with real CSS constraints. GetDesign.md Curated design resources, tools, and references in one place. Good starting point for UX research and visual direction.

Typography & Fonts

Typescale Generate a proportional modular type scale from a base size and ratio. Copy CSS variables directly. Fontpair Curated Google Fonts pairings. Filters by style, mood, and contrast level. Fontjoy ML-powered font pairing generator. Adjustable contrast slider for heading vs body. Fontshare High-quality, free professional fonts from ITF. Alternatives to overused Google Fonts defaults.

Colour Systems

Coolors Fast palette generator with contrast checking, export to CSS variables, and collision detection. Radix Colors 12-step semantic colour scales designed for accessible dark and light modes out of the box. Realtime Colors Preview your palette on an actual UI in real time. Exports Tailwind, CSS variables, and Figma tokens. Color Hunt Community-curated palette library. Filter by mood, season, or popularity.

Icons & Assets

Lucide Consistent 2px-stroke icon library. 1 500+ icons, tree-shakeable, available as React/Vue/Svelte packages. Heroicons Tailwind-authored icon set in outline and solid styles. Zero visual inconsistency within each set. Phosphor Icons 6 weights (thin → bold) from one family. Use one weight throughout and your UI reads as intentional. Squoosh Browser-based image compression. Convert to WebP/AVIF and compare quality vs file size visually.

CSS & Component Libraries

Tailwind CSS Utility-first framework that enforces a design system by default. Prevents arbitrary magic numbers. shadcn/ui Copy-paste accessible components built on Radix primitives. You own the code — no black-box overrides. Open Props CSS custom property design tokens (spacing, easing, shadows, colours). Drop into any project. daisyUI Tailwind component library with 35 built-in themes. Semantic class names instead of utility soup.

Accessibility & Testing

axe DevTools Browser extension that audits WCAG violations inline in DevTools. Zero false positives by design. WAVE Visual accessibility evaluation. Overlays icons on the page showing errors, alerts, and structural issues. PageSpeed Insights Google's Lighthouse runner. Scores performance, accessibility, SEO, and best practices with field data. WebPageTest Deep performance waterfall analysis. Real devices, real networks, filmstrip view of load progression.

Inspiration & Reference

Mobbin Real app UI screenshot library — searchable by screen type, pattern, and app category. Awwwards Jury-selected award site showcasing best-in-class web design and interaction work. Screenlane Mobile UI pattern library. Filter by component type (onboarding, nav, forms) or industry. Refero Searchable design pattern library across web and mobile. Includes interaction recordings.
